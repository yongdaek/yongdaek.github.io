Course Description

This course introduces fundamental concepts in cybersecurity. Students will learn how to think like an adversary, understand cryptographic foundations, network and communication security, anonymity and privacy, usability, and security of emerging cyber-physical and distributed systems. The course emphasizes real-world attacks and defenses, including cellular security, GPS spoofing, and blockchain systems.

Course Modules

  • Introduction and adversary thinking
  • Cryptography and secure communication (symmetric crypto, hash functions, public-key crypto, PKI, TLS)
  • Network security fundamentals
  • Anonymity and privacy
  • Usability and human factors
  • EE Security I: Sensor security, cars
  • EE Security II: Drones, GPS
  • EE Security III: Cellular security
  • EE Security IV: Blockchain and cryptocurrency
